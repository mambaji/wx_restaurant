# wx_restaurant
餐饮小程序前端
## 功能列表

|名称|描述|
|----|----|
|首页|店铺信息展示|
|订单列表|堂吃订单,外卖订单,预约订单|
|个人中心||
|我的地址|地址列表,新增,编辑,滑动删除|
|预约|顾客预约到店时间|
|点餐|店内扫描桌面二维码进行点餐|
|外卖||
|排队|店外扫描二维码排队|
|菜单|店内点餐类型,外卖类型|
|提交订单||
|支付|微信支付,店内点餐支持现金支付|
